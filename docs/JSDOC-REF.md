# Reference for JSDoc Tags

## Types
- @type
- @param (or @arg or @argument)
- @returns (or @return)
- @typedef
- @callback
- @template
- @satisfies

## Classes
- Property Modifiers @public, @private, @protected, @readonly
- @override
- @extends (or @augments)
- @implements
- @class (or @constructor)
- @this


## Documentation
- @deprecated
- @see
- @link


## Other
- @enum
- @author
